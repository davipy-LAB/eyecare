# 👁️ EyeCare - Intelligent Visual Accessibility

**EyeCare** is a digital accessibility platform designed for users with color blindness and eye strain. The project offers a personalized interface that allows you to adjust the visual experience according to each individual's needs, ensuring that the web is a more inclusive place.

## ✨ Main Features

- **🎨 Real-Time Color Blindness Filters**: Application of filters based on color matrices for Protanopia, Deuteranopia, and Tritanopia, correcting visual perception instantly.

 - **🌓 Dynamic Contrast & Comfort Mode**: Fine-tuning of contrast for easier reading and "Comfort Mode," which reduces color harshness to protect vision and prevent eye strain.

- **🌍 Internationalization (i18n)**: Interface with full support for multiple languages ​​(Portuguese, English, and German) and automatic saving of user preferences.

- **📱 Mobile First & Responsive**: Fully optimized design for mobile phones, offering a fluid and accessible experience on any screen size.

- **🚀 Intelligent Onboarding**: Initial setup flow that helps the user define their visual profile right from the first access.

- **🖥️ Modern Dashboard**: Dark mode interface with modern aesthetics and a total focus on usability.

---

## 🛠️ Technologies Used

### Frontend:

- **HTML5 & CSS3**: Modern and responsive structure and styling.

 - **Vanilla JavaScript**: Logic for applying filters and dynamically manipulating the interface.

- **i18next**: Library for language management.

- **SVG Filters**: Technology used for precise color simulation and correction.

### Backend:

- **FastAPI (Python)**: High-performance framework for serving the application.

- **Uvicorn**: Server for running the production environment.

# 👁️ EyeCare - Acessibilidade Visual Inteligente

O **EyeCare** é uma plataforma de acessibilidade digital desenhada para usuários com daltonismo e fadiga ocular. O projeto oferece uma interface personalizada que permite ajustar a experiência visual de acordo com a necessidade de cada um, garantindo que a web seja um lugar mais inclusivo.

## ✨ Funcionalidades Principais

- **🎨 Filtros de Daltonismo em Tempo Real**: Aplicação de filtros baseados em matrizes de cores para Protanopia, Deuteranopia e Tritanopia, corrigindo a percepção visual instantaneamente.
- **🌓 Contraste Dinâmico & Modo Conforto**: Ajuste fino de contraste para facilitar a leitura e o "Modo Conforto", que reduz a agressividade das cores para proteger a visão e evitar o cansaço ocular.
- **🌍 Internacionalização (i18n)**: Interface com suporte total a múltiplos idiomas (Português, Inglês e Alemão) e salvamento automático da preferência do usuário.
- **📱 Mobile First & Responsivo**: Design totalmente otimizado para celulares, oferecendo uma experiência fluida e acessível em qualquer tamanho de tela.
- **🚀 Onboarding Inteligente**: Fluxo de configuração inicial que ajuda o usuário a definir seu perfil visual logo no primeiro acesso.
- **🖥️ Dashboard Moderno**: Interface em modo escuro com estética moderna e foco total na usabilidade.

---

## 🛠️ Tecnologias Utilizadas

### Frontend:
- **HTML5 & CSS3**: Estrutura e estilização moderna e responsiva.
- **Vanilla JavaScript**: Lógica de aplicação dos filtros e manipulação dinâmica da interface.
- **i18next**: Biblioteca para gerenciamento de idiomas.
- **SVG Filters**: Tecnologia utilizada para a simulação e correção precisa de cores.

### Backend:
- **FastAPI (Python)**: Framework de alta performance assíncrona para servir a aplicação.
- **Uvicorn**: Servidor para execução do ambiente de produção.
